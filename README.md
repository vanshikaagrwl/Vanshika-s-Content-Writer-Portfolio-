## 🚀 Getting Started

### 1. Local Testing
Simply open `index.html` in your web browser to view the portfolio locally.

```bash
# On Windows, you can use:
start index.html

# Or open directly in your preferred browser
```

### 2. Deploy Options

#### **GitHub Pages (Free & Recommended)**
1. Create a GitHub repository called `portfolio`
2. Upload these three files to the repository
3. Go to repository Settings → Pages
4. Select "Deploy from a branch" and choose `main` branch
5. Your portfolio will be live at: `https://yourusername.github.io/portfolio/`

#### **Netlify (Free)**
1. Go to [netlify.com](https://netlify.com)
2. Click "New site from Git" or drag and drop your folder
3. Your site will be live in seconds with a Netlify subdomain

#### **Vercel (Free)**
1. Go to [vercel.com](https://vercel.com)
2. Import your repository or drag and drop files
3. Deploy instantly with automatic HTTPS

#### **Custom Domain**
Any of the above services support custom domains. Point your domain's DNS to your hosting provider's nameservers.
