# 🚀 Deployment Guide

Your portfolio is now ready to go live! Here are the easiest deployment options:

## **Option 1: GitHub Pages (Recommended - Free & Easy)**

### Steps:
1. **Create GitHub Account** (if you don't have one): https://github.com/signup
2. **Create New Repository**:
   - Go to https://github.com/new
   - Repository name: `portfolio` (or any name)
   - Make it **Public**
   - Click "Create repository"

3. **Upload Files**:
   - Click "uploading an existing file"
   - Drag and drop or select: `index.html`, `styles.css`, `script.js`, `README.md`
   - Commit changes

4. **Enable GitHub Pages**:
   - Go to repository Settings
   - Scroll to "Pages" section
   - Under "Source", select `main` branch
   - Click "Save"

5. **Your portfolio is live at**: `https://yourusername.github.io/portfolio/`

---

## **Option 2: Netlify (Free - Super Easy)**

### Steps:
1. Go to https://netlify.com
2. Click "New site from Git" or drag & drop your folder
3. Select your repository (or drag files)
4. Your site is deployed instantly!
5. Free URL: `https://randomname.netlify.app`
6. Connect custom domain anytime

---

## **Option 3: Vercel (Free - By Next.js creators)**

### Steps:
1. Go to https://vercel.com
2. Click "New Project"
3. Upload or import your repository
4. Deploy instantly
5. Get a free URL and custom domain support

---

## **Option 4: Custom Domain**

After deploying to any of the above:
1. Buy a domain (GoDaddy, Namecheap, Google Domains, etc.)
2. Point domain to your hosting provider
3. Connect in platform settings
4. Your portfolio is at: `https://yourname.com`

---

## **Local Development Server**

To run locally:
```bash
node server.js
```
Visit: http://localhost:8000
